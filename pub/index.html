Swish.js

Create interactive basketball shot charts and simulation tools with this small javascript library

Examples
Basic halfcourt shot charts

Plot basketball actions on a basketball halfcourt.

Set the desired width and choose "half" for a halfcourt. Pass in data containing the type of the action, and the x and y coordinates.

Action types have unique colours and can include: make, miss, pass, dribble, rebound, steal, block, turnover, and foul. New types will be given a random colour.

The size of each action will be scaled to be proportional to the set width of the court.



JS 
const s1 = swish();
s1.drawCourt(1, 600, "half", "example1Court");
s1.drawActions(data1);

Fullcourt shot charts with teams

Plot basketball actions on a basketball fullcourt with two different teams.

Set the desired width and choose "full" for a fullcourt. Pass in data containing the type of the action, the x and y coordinates, and the teams.

The two different teams will have different shapes.



JS 
const s2 = swish();
s2.drawCourt(2, 400, "full", "example2Court");
s2.drawActions(data2);

Create popups for each action

Create popups to display the team, player, and action type for each action.

Pass in data containing the type of the action, the x and y coordinates, teams, and players.

Hover over each action on the court to see the popup. Hover over the base of the arrows to see their popups.



JS 
const s3 = swish();
s3.drawCourt(3, 400, "full", "example3Court");
s3.drawActions(data3);
s3.createPopup();

Create a slider to show and hide actions

Create a slider to show and hide each action over time.

Pass in data containing the type of the action, the x and y coordinates, teams, players, periods, and times.

Drag the slider to show and hide actions. The current action is highlighted with a blue outline.



JS 
const s4 = swish();
s4.drawCourt(4, 600, "half", "example4Court");
s4.drawActions(data4);
s4.createSlider("example4Court");
HTML 
<‍div id="example4Court"><‍/div>

Create a timeline

Create a timeline to show the times and descriptions of each action.

Set the desired width and height of the timeline. Pass in data containing the type of the action, the x and y coordinates, teams, players, periods, times, and descriptions.

Drag the slider to show and hide actions and update the timeline. The timeline contains the period, time and description of the action. The current action is also highlighted in the timeline.



JS 
const s5 = swish();
s5.drawCourt(5, 400, "full", "example5Court");
s5.drawActions(data5);
s5.createSlider("example5Court");
s5.createTimeline(200, 400, "example5Timeline");
HTML 
<‍div id="example5Court"><‍/div>

Customize colours and shapes

Change the colours for action types and shapes for teams.



JS 
const s6 = swish();
s6.drawCourt(6, 600, "half", "example6Court");
s6.drawActions(data6);
s6.setColour("rebound", "blue");
s6.setShape("team1", "square");
s6.setShape("team2", "circle");
s6.createPopup();
